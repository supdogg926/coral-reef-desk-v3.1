# North Star UI Reference v0.3

The current high-fidelity reference image is the ReefIdle V3 North Star UI Reference v0.3.

## Locked Direction

Future UI work must iterate toward this reference instead of freely redesigning the interface.

- The display tank is the main visual focus.
- The sump shows the equipment system and chamber structure.
- The bottom area uses an Apple-inspired control console style.
- The interface should feel like a semi-realistic reef system dashboard, not a cartoon or 3D game view.
- Codex tasks should implement the reference progressively and should not invent unrelated UI layouts.

## Implementation Rule

When adding UI, preserve the current direction: display tank first, visible sump equipment second, compact control/status console third.
