# Core Game Loop

ReefIdle V3 is built around a real reef aquarium loop: build equipment, stabilize the system, increase livestock capacity, and use reef points to unlock deeper system tiers.

## Loop Summary

1. Player places equipment into the display tank, cabinet, sump, or utility areas.
2. Equipment increases system stability, filtration capacity, flow, heating control, nutrient export, and automation coverage.
3. Higher stability increases livestock carrying capacity and reduces maintenance pressure.
4. Livestock generates observation value and reef points over time.
5. Reef points unlock new equipment, new livestock, larger capacity, and more advanced reef system functions.
6. Tier 1 grows into Tier 2, then Tier 3, with each tier adding stronger stability tools and higher system ceilings.

## Primary Player Motivation

The player is not only collecting animals. The player is building a believable marine aquarium system that can safely support more sensitive livestock over time.

## Core Resources

- Reef points: idle progression currency generated by stable livestock display value.
- System stability: derived from equipment quality, placement coverage, and water chemistry control.
- Livestock capacity: derived from tank size, filtration, oxygenation, flow, and stability.
- Maintenance pressure: operational cost and failure pressure created by livestock load and weak equipment.

## Tier Growth

- Tier 1: basic life support and first stable display system.
- Tier 2: improved control, cooling, sterilization, flow, and refugium efficiency.
- Tier 3: automation and advanced reef chemistry control for higher-value livestock.
